FSAE SPONSOR PROJECT

A sponsor project for Illinois Motorsports that completed FSAE (Formula Society of Automotive Engineers) Michigan and FSAE Lincoln autocross tracks in 3D and exported to Assetto Corsa racing simulator. Group of 5 engineers utilized Unity for modeling and Assetto Corsa SDK for game integration to help Illinois Motorsports team to prepare using Oculus VR headset. Monocular depth cue and stereopsis has been incorporated on our textured tracks to create comfortabel and realsitic racing simulation, and physics engine were taken care of by Assetto Corsa SDK.



FLIGHT SIMULATOR

This machine problem requires to set up the terrains in Unity, import a model for the plane, create several scenes for start menu with ray casting, interactive tutorial with pitch, yaw and roll controls, and make it a shooting game with sound and score display.
